HUSTLE

let's get moving!

I chose the verb hustle based on the adjective busy. As you click, the zigzags appear to be rushing forward, like they are in a hurry. Eventually, the canvas will fill with the zigzags. The user has made it just in time! I used the Basic Arithmetic guide in the MAKE book to add the rectangles on top, which gives the feel of forward motion. Using an else statement, I added a the "hurry up" text to remind the user to keep moving. Once the user fills the canvas with zigzags, the message will be replaced, indicating that the state has changed.
