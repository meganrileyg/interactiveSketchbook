Hesitation

The phone is ringing!

Click the "]" key to reach for the phone.

Currently, the circle is set to expand as the "phone" rings. The main variable is the action keyPressed "]" which triggers the circle to grow.

The pressure is accumulative. The circle will continue to grow as the "phone" rings, and the user continues to press the "]" key. There is a random "drop call" that triggers 2.5% of the time, causing the circle to restart, effectively delaying the user from "answering" the phone.

The user must wait 10 seconds to have a "conversation" with the person on the other end of the line.
